 A control API for a novel modular synthetic muscle, Myo\+Syn is a portmanteau of Synthetic (as in man-\/made) and Myosin (a protein fundamental to muscle function, which is also how you pronounce Myo\+Syn). This C++ API allows us to actuate this muscle with closed-\/loop control using our custom Quick\+DAQ software.\hypertarget{index_autotoc_md1}{}\doxysection{Dependencies}\label{index_autotoc_md1}
